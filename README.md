# Football Explorer

Football Explorer is a web application built with HTML,CSS and JavaScript that allow users to search for football teams and players using a public football API.

## Features

- Search football teams
- Search players
- View team information
- Team badges and player photos
- Fast API requests
- Responsive design
- Enter key support
- Dark mode

##Technologies

- HTML
- CSS
- JavaScript (ES6)
- Fetch API
- TheSports API
